# Security Policy

## Supported Versions

Aegis Hub is currently in an active development phase. 
Only the latest branch/commit on the `main` branch is actively supported with security patches.

## Reporting a Vulnerability

Security is the absolute highest priority for this project. 
If you discover a vulnerability, **do not open a public GitHub issue**.

Please report any security issues privately by emailing the repository owner directly, or by using the GitHub Security Advisory private reporting feature if enabled on this repository.

We will acknowledge receipt of your vulnerability report within 48 hours and strive to send you regular updates about our progress. If the vulnerability is accepted, we will issue a patch as quickly as possible.
